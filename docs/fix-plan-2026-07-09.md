## Nhận xét kiến trúc lớn nhất (từ review — ĐỒNG Ý)

> Chưa có **text transaction model**. Engine đang trộn preedit + commit + reset.

Đây là nợ gốc. Các bug click-mất-chữ, backspace chậm, smart-mode conflict đều dễ giải hơn nếu tách:

```rust
/// Trạng thái một phiên gõ, tách bạch 4 tầng
struct Transaction {
    raw: String,        // phím thô user đã gõ: "tieeng"
    display: String,    // đang hiển thị (preedit hoặc đã bắn): "tiếng"
    committed: String,  // phần đã chốt, không sửa nữa
    cursor: usize,
}
```

Kèm **undo stack**: mỗi keystroke push snapshot → backspace = `pop()` O(1), thay vì reparse.
Refactor này là P0-2 vì nó đồng thời fix backspace VÀ dọn state machine cho mọi việc sau.
